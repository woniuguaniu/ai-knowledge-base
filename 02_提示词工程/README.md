## 学习建议

### 给完全零基础的你

1. **第1周**：学习模块 01-02，建立正确心智模型，练会基本写法
2. **第2周**：学习模块 03-04，掌握示例、角色和推理技巧
3. **第3周**：学习模块 05-06，把复杂任务拆开并落地到工作场景
4. **第4周**：学习模块 07-08，进入编程场景和模板化复用
5. **第5周**：学习模块 09-10，建立优化评估能力，理解现代 AI 应用架构

### 核心学习方法

- **动手实践**：每个技巧都在 ChatGPT / Claude / Gemini 上实际试一次
- **对比学习**：同一个任务，尝试不同提示写法，对比输出差异
- **建立模板**：把验证过的好提示词沉淀成自己的模板
- **持续迭代**：提示词没有标准答案，持续优化比一次写对更重要
