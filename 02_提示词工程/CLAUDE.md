# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概述

这是一个**提示词工程（Prompt Engineering）学习指南**，纯文档项目，无代码构建/测试流程。所有内容为 Markdown 格式的中文教程，面向从零基础到高级的学习者。

## 内容结构

项目当前包含 **10 个递进式学习模块**（`01_` ~ `10_`），并按 README 的学习路线分为 4 个阶段：

- **第一阶段：认知建立**
  - `01_AI对话的底层逻辑.md`：AI 工作原理、Token、参数、模型差异

- **第二阶段：核心技能**
  - `02_写好第一个提示词.md`：指令写法、约束、分隔符、输出格式控制
  - `03_用示例教会AI.md`：少样本、角色提示、系统提示词、上下文控制
  - `04_让AI深度思考.md`：CoT、自我一致性、回溯提示、元提示
  - `05_复杂问题的拆解策略.md`：ToT、ReAct、提示词链、任务分解

- **第三阶段：场景实战**
  - `06_工作效率场景.md`：写作、数据分析、办公效率、学习教育
  - `07_编程与技术场景.md`：代码生成、代码审查、Bug 调试、重构

- **第四阶段：专业进阶**
  - `08_提示词模板与框架.md`：经典框架、模板设计、提示词库
  - `09_系统化优化与评估.md`：PDCA、A/B 测试、调试技巧、评估框架
  - `10_AI应用架构.md`：RAG、Agent、多模态提示、提示词安全、多 Agent 协作

附有 `速查手册.md`（快速参考）和 `练习题集.md`（分级练习）。

## 编辑规范

- 所有文档使用中文撰写
- 文件按编号顺序组织，修改时保持现有编号体系
- 使用 Markdown 格式，保持与现有文档一致的标题层级和排版风格
- `B站提示词工程/` 目录存放配套视频相关素材
